# Dataset description

The dataset is comprised of a single alignment file that has been processed from 462 whole genome isolates of *Klebsiella pneumoniae* from patients at a long-term acute care hospital (LTACH) to only retain the parts of each genome where they differ from the reference. This is generated by the pipeline described below, taken from the  in the paper by Hawken et al. (2022)[^1]:

> All whole genomes first undergo QC with FastQC and Trimmomatic, and thenn raw reads are mapped to sequence-type (ST) specific reference genomes using BWA-MEM algorithm from BWA. Variant calling is performed with samtools. Reference genomes with matching sequence types (STs) to study strains are chosen to maximize core-genome  orthologous regions for SNV identification and decrease erroneous variant calls. Consensus alignments generated from read mapping to ST specific reference genomes undergo detection and masking of putative recombinant regions using gubbins. Lastly, reference genome annotations are generated using Prokka version 1.14.5 and variant annotations are predicted using snpEff version 4.3t.

[^1]: Hawken, Shawn E., et al. “Threshold-Free Genomic Cluster Detection to Track Transmission Pathways in Health-Care Settings: A Genomic Epidemiology Analysis.” The Lancet Microbe, vol. 3, no. 9, Sept. 2022, pp. e652–62. DOI.org (Crossref), https://doi.org/10.1016/S2666-5247(22)00115-X.
